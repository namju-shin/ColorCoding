VISUAL DESIGN - 공익광고

우리말 바로잡기를 위한 공익광고

외래어보다 우리말을 더 많이 사용하자는 취지로 제작한 공익광고입니다.
일상에서 자주 사용하는 외래어 대신, 대체할 수 있는 우리말을 소개하여
실생활에서 자연스럽게 사용할 수 있도록 표현하였습니다.
친숙한 영상 연출과 직관적인 디자인을 통해 누구나 쉽게 받아들이고 공감할 수 있도록 제작하였습니다.

프로그램 정보
◇ 제작기간 : 약 열흘
◇ 사용 Tool
- Illustrator / Photoshop → 비주얼 디자인 제작
- After Effect → 영상 편집 & 효과
- 네이버 클로바 더빙 → 나레이션 음성 제작

VISUAL DESIGN - 영상CG작업

영상CG작업

좋아하는 애니메이션을 주제로 한 소개영상입니다.
애니메이션의 주요 설정을 더욱 직관적으로 전달할 수 있도록 구성하고,
주인공 캐릭터들의 외형을 더욱 귀엽고 개성 있게 표현하는 데 집중했습니다.
감각적인 연출과 생동감 있는 나레이션을 더해, 원작의 매력을 효과적으로 살리고자 하였습니다.

프로그램 정보
◇ 제작기간 : 약 1주
◇ 사용 Tool
- Illustrator / Photoshop → 비주얼 디자인 제작
- After Effect → 영상 편집 & 효과
- Typecast → 나레이션 음성 제작
